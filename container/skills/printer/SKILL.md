# Printing to Kyocera TASKalfa 3252ci

The CUPS socket is mounted at `/workspace/extra/cups/cups.sock`. All print commands must use this socket.

## Environment setup

Every print command must be prefixed with the CUPS server path:

```bash
export CUPS_SERVER=/workspace/extra/cups/cups.sock
```

Or prefix each command:

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock lp ...
```

## Printer name

The configured printer is: `Kyocera-TASKalfa-3252ci-KPDL`

## Commands

### Check printer status

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock lpstat -p
```

### Print a file

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL /path/to/file
```

### Print with options

```bash
# Double-sided (long edge)
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -o sides=two-sided-long-edge file.pdf

# Multiple copies
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -n 3 file.pdf

# Specific pages
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -o page-ranges=1-5 file.pdf

# Landscape
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -o landscape file.pdf

# Fit to page
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -o fit-to-page file.pdf

# Color mode (mono for black & white)
CUPS_SERVER=/workspace/extra/cups/cups.sock lp -d Kyocera-TASKalfa-3252ci-KPDL -o ColorModel=Gray file.pdf
```

### List print queue

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock lpstat -o
```

### Cancel a print job

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock cancel <job-id>
```

### Show printer options

```bash
CUPS_SERVER=/workspace/extra/cups/cups.sock lpoptions -d Kyocera-TASKalfa-3252ci-KPDL -l
```

## Supported file types

The printer accepts: PDF, PostScript, plain text, PNG, JPEG, TIFF. For other formats, convert to PDF first using available tools.

## Tips

- Always check printer status before printing to confirm it's online
- For documents generated by agent-browser (`agent-browser pdf output.pdf`), print the resulting PDF directly
- The printer is on the local network at 172.16.17.50 — no internet required
